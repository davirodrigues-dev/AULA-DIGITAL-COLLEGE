# INTRODUÇÃO AO CSS

CSS (Cascading Style Sheets, ou Folhas de Estilos em Cascata) é a linguagem utilizada para definir o estilo visual de documentos HTML, controlando o layout, cores, fontes, espaçamentos e responsividade.

## Por que usar CSS? 

O CSS separa o conteúdo (HTML) da apresentação. Permite deixar páginas mais atrativas e organizadas, melhorando a experiência do usuário.

## Forma de aplicar CSS:

Inline: direto no elemento, usando o atributo style.
Interno: bloco <style> dentro do elemento <head>.
Externo: arquivo .css separado (recomendado para projetos reais.)
